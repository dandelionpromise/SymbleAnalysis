========================================================================
    控制台应用程序：SymbleDebug 项目概述
========================================================================

应用程序向导已为您创建了此 SymbleDebug 应用程序。

本文件概要介绍组成 SymbleDebug 应用程序的每个文件的内容。


SymbleDebug.vcxproj
    这是使用应用程序向导生成的 VC++ 项目的主项目文件，
    其中包含生成该文件的 Visual C++ 
    的版本信息，以及有关使用应用程序向导选择的平台、配置和项目功能的信息。

SymbleDebug.vcxproj.filters
    这是使用“应用程序向导”生成的 VC++ 项目筛选器文件。 
    它包含有关项目文件与筛选器之间的关联信息。 在 IDE 
    中，通过这种关联，在特定节点下以分组形式显示具有相似扩展名的文件。
    例如，“.cpp”文件与“源文件”筛选器关联。

SymbleDebug.cpp
    这是主应用程序源文件。

/////////////////////////////////////////////////////////////////////////////
其他标准文件：

StdAfx.h，StdAfx.cpp
    这些文件用于生成名为 SymbleDebug.pch 的预编译头 (PCH) 文件和
    名为 StdAfx.obj 的预编译类型文件。

-------------------------------软件需求--------------------------------------
--开放的外部接口：
-初始化函数
-SYMBLE_API char *SYM_Init(char *FileName)
-----传入参数：cfg.csv文件目录\文件名.csv
-----返回参数：文件级名，格式为：文件名$unfold,文件名$unfold,……文件名$unfold,#
-----作用：	1、导入符号库（.csv）；
------------2、解析.c文件名并返回所有.c文件名。
-符号展开函数
-SYMBLE_API char *SYM_SymbolUnfold(char *SymName)
-----传入参数：需要展开的当前级符号名,若为多个,格式为：当前级符号名,当前级符号名,……当前级符号名#
-----返回参数：传入符号下一级的兄弟姐妹，格式为（结构体、联合体、数组均以此方式展开）：传入符号下一级符号名$unfold,传入符号下一级符号名$unfold,……传入符号下一级符号名$unfold#
-----作用：	1、解析传入符号下一级的符号并返回所有传入符号下一级的符号。
-符号属性获取函数
-SYMBLE_API char *SYM_GetSymbolAttribute(char *SymNameAttr)
-----传入参数：符号名$需要获取的属性,符号名$需要获取的属性,……符号名$需要获取的属性#或符号名$需要获取的属性$需要获取的属性$……需要获取的属性#
-----返回参数：传入符号下一级的兄弟姐妹，格式为（结构体、联合体、数组均以此方式展开）：传入符号下一级符号名$unfold,传入符号下一级符号名$unfold,……传入符号下一级符号名$unfold#
-----作用：	1、解析传入符号下一级的符号并返回所有传入符号下一级的符号。
-解码函数
-SYMBLE_API char *SYM_BigDecoding(char *SymAddrSizeData)
-----传入参数：地址$长度$需要解码的数据,若为多个,格式为：地址$长度$需要解码的数据,地址$长度$需要解码的数据,……地址$长度$需要解码的数据#
-----返回参数：符号名$符号值(NULL为可以展开的符号，没有值),符号$符号值,……符号$符号值#
-----作用：	1、解码下位机上传的数据；
-从大包内提取小包的解码函数
-SYMBLE_API char *SYM_SmallDecoding(char *SymAddrSizeData)
-----传入参数：地址$长度$需要解码的数据,若为多个,格式为：地址$长度$需要解码的数据,地址$长度$需要解码的数据,……地址$长度$需要解码的数据#
-----返回参数：符号名$符号值(NULL为可以展开的符号，没有值),符号$符号值,……符号$符号值#
-----作用：	1、解码下位机上传的数据；
-编码函数
-SYMBLE_API char *SYM_Encoding(char *SymNameVal)
-----传入参数：符号$符号值,若为多个，格式为：符号$符号值,符号$符号值,……符号$符号值#
-----返回参数：地址$长度$数据,地址$长度$数据,……地址$长度$需要解码的数据#
-----作用：	1、编码上位机下传的数据；
--注：传递参数的格式:参数内不同项以“$”分隔、不同参数以“,”分隔、末尾以#结束
